# PUDDiT – Software modeling and analysis

## Инсталация

1. Създайте база данни `PUDDiT` в MS SQL Server.
2. Изпълнете `schema.sql` върху тази база (създава таблици, ключове, функция, процедура и trigger).
3. Изпълнете `seed.sql`, за да напълните базата с примерни данни (потребители, канали, постове, коментари, гласове и доклади).
4. Използвайте `queries.sql` за да изпробвате всички заявки.
